## Módulos 2, 3, 4, 5, 6, 10 e 11
Esta atividade consolida habilidades e conceitos trabalhados ao longo de diversos módulos do curso de Unity Iniciante, por meio do desenvolvimento de um quiz interativo.
Durante o projeto, foram aplicados conceitos de gerenciamento de cenas, permitindo a navegação entre as telas do jogo, incluindo a transição para a tela de resultados ao final do quiz.
Também foi realizado o processo de importação de assets da Unity, com destaque para arquivos de áudio, que foram vinculados a GameObjects e controlados por meio de scripts em C#.
Foram importadas fontes personalizadas do Google Fonts, aplicadas à interface do projeto.

O desenvolvimento envolveu:
- Serialização de dados para organização e controle das informações do quiz
- Gerenciamento de cenas e áudio
- Uso de PrayerPrefs para salvar informações como score do jogador.
- Uso de eventos OnClick para interação com botões  
- Manipulação de elementos de interface (UI)  
- Aplicação prática de lógica de programação em C# dentro do contexto da Unity 

Esse projeto representa a integração de múltiplos conhecimentos adquiridos ao longo do curso, reforçando a compreensão dos principais recursos da Unity e sua aplicação em um projeto funcional.
O package da cena e suas dependências pode ser encontrado em [CompiledModules](https://github.com/juliaazeved0/Mentorama-Unity-Beginner/tree/main/CompiledModules)
